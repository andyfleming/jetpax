# Jetpax

Pragmatic development environment manager (intended for supporting a local development environment).


## Prerequisites

* Docker (v18+ recommended)
* Node (v10 recommended)

## Installing

Install the package globally

```bash
npm install -g jetpax
```

Start the server

```bash
jpx up
```

Open the UI

```bash
jpx ui
```
